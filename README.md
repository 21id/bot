# Open Source Identification system for School 21

[21ID Telegram bot](https://t.me/id_21bot) is an identification system, 
developed by Tashkent and Samarkand campus students - to make process of finding 
other peers and joining School 21 communities easier


## Contributing
We'd love for you to create **pull requests from the development branch (develop)**. 
Latest releases are created from the _main branch_.

You can find our plans for the future releases, needed fixes, mentioned bugs and 
other things in the repository
[Project section](https://github.com/orgs/21id/projects/2)

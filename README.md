# Open Source Identification system for School 21 students

21ID is an identification system, developed by Tashkent and Samarkand campus 
students - to make process of finding other peers and joining School 21 communities 
easier


## Contributing
We'd love for you to create pull requests from the development branch. Latest 
releases are created from the main branch.
